STMng — Fingerprints charts window

This window shows seven different kinds of charts:

Fingerprint
Fingerprint for the selected structure step.
Energy
Chart of per atom energy for all structures.
Energy-Dist
Chart of per atom energy differences from the minimum energy versus the distance from the point of minimal energy.
Hist energies
Histogram of energies.
Hist distances
Histogram of distances.
Order
Order parameter for all structures.
Distances
Distances from the given fingerprint.

Besides the chart kind selector, the window has the following controls:

Structure step selector slider Select the structure step for which the fingerprint or the distances should be charted. This control appears only when fingerprint or distances chart are selected.
Bin count slider Set the number of bins for the histograms. This control appears only when one of the two histograms are selected.
Close button Close the window.
