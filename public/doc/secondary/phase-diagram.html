STMng — Variable composition phase diagram window

The window shows two kind of diagrams for variable composition sets: the detailed and the summary views. The kind of diagram could be selected using the “Show summary” toggle.

Detailed view

This diagram shows for each structure type the interval of pressures for which it lies on the convex hull. The range of pressures goes from -200 GPa to 200 GPa in steps of 0.1 GPa.

Hovering on each strip show the formula, the corresponding step in the input set and the range of pressures for which this structure is stable.

The vertical gaps in the strips signal the change of structure: different step but same composition.

Summary view

This diagram combine the various intervals to have one line for each base composition. For example, if the end members are “Cu” and “S”, the compositions “Cu”, “Cu3” and “Cu7” are merged into a single “Cu” line. As before the range of pressures goes from -200 GPa to 200 GPa in steps of 0.1 GPa.

Hovering on each strip show the formula and the range of pressures for which this structure is stable.
