STMng — Fingerprints results export window

This window collect the export commands for some of the fingerprints results:

Export results
Export the structures selected without duplicates to a POSCAR file and a corresponding energy per structure or atom file. The entries are sorted on increasing energy values. If no energy has been loaded, the energy file is not created and the structures are unsorted.
Min. energy per group
Select one point per group that has minimal energy.

Note that these functions are the same found in the Fingerprint scatterplot window.

When selecting one of these export functions, a file selector appears to select a POSCAR file. If energies are present, a corresponding file with extension .energy will be created.

If energies are present, a toggle appears to select if the energy saved should be per atom and not per structure.

Note: This windows enables exporting results without incurring the high cost of projecting fingerprints to 2D before visualizing the scatterplot.
