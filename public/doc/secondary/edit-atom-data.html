STMng — Edit atom data dialog

Each atom has some characteristics, like color and covalent radius, stored in a default set. This set is used by the Draw structure node and by the Compute bonds one.

With this editor a new set of values could be created and used. In the main menu the set to use could be selected by a checkbox.

Note that the changes are not reactive, that is, they do not change what has been already visualized. Their effects appear, for example, at the next structure load.

The specie to edit is selected clicking on the cards on the left. Note that the circle size is proportional to the atom covalent radius.

The Reset button discards all modifications done and reload the default data set. The Save button save modifications as the user data set. The Close button asks confirmation before closing the dialog if there are unsaved changes.

The dialog subtitle lists which set is loaded and if it has been modified.
