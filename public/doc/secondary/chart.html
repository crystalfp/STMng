STMng — X-Ray diffraction chart

The window show the diffraction spectra and the hkl plus multiplicity peak values for the current structure.

Show grid toggle Show or hide the axis grid in the chart.
Save peaks button Open the file selector to export the diffraction spectra 2θ, intensity and hkl plus multiplicity peak label values. The peaks are one per line in three columns.
Save points button Open the file selector to export the chart X, Y values. The points are one per line in two columns.
Save image button Export the chart as a PNG image. Note that the image is exactly what is on screen. So, for example, to obtain an image with white background simply deselect the “Dark theme” in the View menu.
Close button Close the window.
