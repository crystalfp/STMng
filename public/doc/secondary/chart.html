STMng — X-Ray diffraction chart

The window show the diffraction spectra and the hkl plus multiplicity peak values for the current structure. Note that hovering the mouse on the chart show the corresponding spectra value.

This window is created to stay always on top of other windows.

Save points button Open the file selector to export the chart X, Y values. The points are one per line in two columns.
Save image button Export the chart as a PNG image.
Transparent toggle Use a transparent background for the chart image.
Close button Close the window.
