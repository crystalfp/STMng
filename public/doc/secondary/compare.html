STMng — Compare structures window

Shows pair of structures and make possible rotate them to visually compare them. The structures are shown with coincident origins.

Compare selected — left list Select the step of the first structure to compare. The structure and its unit cell will be colored blue as the list header color. Selecting again an entry deselects it. The same structure cannot be selected twice.
Compare selected — right list Select the step of the second structure to compare. The structure and its unit cell will be colored green as the list header color. Selecting again an entry deselects it. The same structure cannot be selected twice.
Remove — left button Remove the selected structure step from the left list. The selection will be updated also in the scatterplot window.
Remove — right button Remove the selected structure step from the right list. The selection will be updated also in the scatterplot window.
Reset button Reset the rotation angles.
Around a, b, c slider Set the rotation angles around the corresponding basis vector in the left structure.
Close button Close the window.
