STMng — Fingerprints scatterplot window

The scatterplot window shows the fingerprints as points in a 2D space. The points are colored by the associated value.

This window shows five different kinds of scatterplot:

Group
One point for each structure colored by group.
Energy
One point for each structure colored by energy.
Fidelity
One point for each distance with X value the distance and Y value the projected distance. More the points live near the diagonal, more the projection preserves the original distance. Note that the colorscale goes from white for distances that have a perfect mapping, to black for badly mapped distances. Note also that, for performance reasons, the number of points is reduced to less than 20,000.
Quality
Grouping quality using the silhouette coefficients. These coefficients goes from 1, that show perfect association of the point to the group, to -1, that show that the point is in the wrong group. There are two special cases: 1) the point is alone in a group and 2) the point is on the border between two groups.
From min
One point for each structure colored by its distance from the point with minimum energy.

The grouping quality color scale is:

  • ⬚ Strong (silhouette > 0.7)
  • ⬚ Reasonable (silhouette > 0.5)
  • ⬚ Weak (silhouette > 0.25)
  • ⬚ Bad (silhouette > 0.05)
  • ⬚ Very bad (silhouette < -0.05)
  • ⬚ Cannot measure (group of one element)
  • ⬚ On the border between two groups (silhouette between -0.05 and 0.05)

Clicking on one point select it and show a small popup.

Dragging the mouse with the right button down makes a rectangular selection that selects all points inside. If done with the control key pressed, deselects all selected points inside the selection.

Besides the scatterplot kind selector, the scatterplot window has the following controls:

Point radius slider Radius of the scatterplot points.
Deselect button Deselect all selected points.
Show legend toggle Show a legend for the current type of plot.
Close button Close the window.

The left side of the window contains controls to manage point selection.

Select all button Select all points.
Deselect all button Deselect all selected points.
Group slider Select one of the groups present.
Select group button Select all points pertaining to the selected group.
Compare selected button Open the window to visually compare pairs of structures. The function is available when at least two points are selected in the scatterplot.
Export selected button Export the structures whose points have been selected to a POSCAR file and a corresponding energy file. The entries are sorted on increasing energy. If no energy has been loaded, the energy file is not created and the structures are unsorted.
