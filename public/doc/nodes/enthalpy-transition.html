Enthalpy transitions

Compute enthalpy transitions under pressure changes

The structure enthalpy increases with pressure this way: H = E + PV where H is the enthalpy (eV), E the structure energy (eV), P the pressure (GPa) and V the cell volume (Å³).

This node find the structures more stable when pressure increases and list them.

Controls

Accumulated structures

Count readonly text Count of accumulated structures. The start and stop of the accumulation are done from the Structure Reader page.
Reset button Clear the accumulated structures.

Results

The enthalpy transition structures data table has the following columns:

StepThe structure step in the input sequence.
FormulaThe structure formula.
Pressure (GPa)The pressure from which this structure is the most stable one.

Save results

Save transition structures button Show the output file selector to save the analysis output.
Output file file selector The file selector selects the POSCAR file that will contains the transition structures. Then a second file with same name and path but extension changed to “.energy” is created with the structure energies. The saved energies are per structure.
