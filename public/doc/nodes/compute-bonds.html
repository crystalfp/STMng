Bonds

Add bonds to the input structure

Compute bonds based on interatomic distances multiplied by a fraction of the sum of covalent radii.

Controls

Enable compute bonds toggle Enable bonds computation.
Bonding min distance slider Minimum distance between atoms to create a bond.
Bonding max distance slider Maximum distance between atoms to create a bond.
H Bonding max distance slider Maximum distance between atoms to create a hydrogen bond.
H Bonding max valence angle slider Maximum valence angle to create a hydrogen bond. In X–––H···Y the valence angle is HX̂Y
Multiplier per atom pair toggle If not set, there is only one multiplier for all atoms pairs sum of covalent radii. Otherwise, a table appears with an entry for each possible atom types pair with different multipliers.
Add bonded atoms outside unit cell radio buttons Select how to continue the bonds outside the unit cell.
None
No bonds are added outside the unit cell.
Neighbors
Atoms are added outside the unit cell that have a bond to some inside atom (default).
Full
Atoms are added outside the unit cell until a bond exists with an atom inside the cell or with an atom outside the cell that have a bond with an atom connected to an inside atom. This method is useful for molecular crystals.
Poly
Atoms that could form a polyhedra with at least one internal atom and one or two outside atoms are added outside the unit cell.
Reset parameters button Reset parameters to the default values.
