Measures

Description

By clicking on one to three atoms (using Ctrl-Click), this module provides:

Atoms type and their coordinates. These lines are colored like the atoms selection marker.
Distances between pair or atoms.
Angle (in degrees) created by the three atoms.

The information provided depends on the number of atoms selected. If more than three atoms are selected the first entered will be deselected.

Selecting again a selected atom deselects it.

Parameters

Deselect button Deselect selected atoms.
