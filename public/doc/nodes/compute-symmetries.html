Symmetries

Compute and apply symmetries

Apply the symmetries present in the input structure using the SgInfo library. Then search for symmetries using the Spglib library. This step could also standardize the structure cell and compute the Point group of the output structure.

Controls

Apply input symmetries toggle Apply or ignore the symmetries contained in the input structure. This toggle is disabled if no input symmetry is present.
Enable find symmetries toggle Search for symmetries.
Standardize cell toggle Standardize the structure cell.
Only standardize cell toggle Standardize the structure cell without searching for symmetries.
Primitive cell toggle Try to find a primitive cell instead of a conventional one.
Standardize cell tolerance slider Tolerance for the standardization process.
Find symmetries tolerance slider Tolerance for the search of symmetries.
Symmetry display mode radio buttons Select how to visualize the symmetries on output. The input symmetries obey the chosen representation if it is possible. This control is hidden if no display of symmetries are present.
Intl.
Show the international symbol.
SymmOp
Show the final symmetry as a list of symmetry operations.
Table
Show the space group type number as defined in the International Tables for Crystallography (ITA).
Input symmetry readonly text Input structure symmetry. The field is hidden if no input symmetry is present.
Final symmetry readonly text Output structure symmetry. The field is hidden if “Enable find symmetries” is not set.
Fill unit cell toggle Copies atoms that are on the cell borders and sides to the other side of the cell.
Fill unit cell tolerance slider Tolerance used when checking if an atom is on the boundary of the cell.
Compute point group toggle Compute point group (Schoenflies symbol) for the output structure.
Position tolerance slider Distance tolerance to consider sites as symmetrically equivalent. This control appears only when the “Compute point group” toggle is set.
Eigenvalues tolerance slider Tolerance to compare eigen values of the inertia tensor. This control appears only when the “Compute point group” toggle is set.
Point group readonly text The Schoenflies symbol of the detected point group. This output appears only when the “Compute point group” toggle is set.
Show symmetries dialog button Open a window to show input and output symmetries.

Credits

Apply symmetries uses the SgInfo library.

Find symmetries and standardize cell use the Spglib library.

The point group analyzer is the porting of the Pymatgen class PointGroupAnalyzer.
