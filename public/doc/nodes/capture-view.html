Capture

Capture screen content

Capture the screen content as a snapshot in an image or PDF file or the 3D objects as a STL model. Captures also a sequence of screens in a movie file.

Controls

Format radio button Selects the format of the saved image file: PNG, JPEG or PDF.
Transparent background toggle Change the background to transparent when capturing a PNG snapshot.
Capture snapshot button Capture the viewer content and write it to a selected file in the given format.
Start/Stop recording button Capture a sequence of screenshots and at the end of the capture write them to a selected file as a movie in WEBm, MP4, OGG or MKV format.
Format radio button Selects the format of the saved STL file: ASCII or Binary.
Capture geometry button Capture the current viewer content and write them to a selected STL format file. Note: currently the STL file captures only atoms and bonds, not other geometries like polyhedra or a backbone tube.
