Capture

Capture screen content

Capture the screen content as a snapshot in a image file or STL model. Or capture a sequence of screens in a movie file.

Controls

Format radio button Selects the format of the saved image file: JPEG or PNG.
Capture snapshot button Capture the viewer content and write to a selected file in the given format.
Start/Stop recording button Capture a sequence of screenshots and at the end write them to a selected file as a movie in WEBm, MP4 or AVI format.
Format radio button Selects the format of the saved STL file: ASCII or Binary.
Capture geometry button Capture the current viewer content and write them to a selected STL format file.
