Capture

Capture screen content

Capture the screen content as a snapshot in an image or PDF file or the 3D objects as a STL model. Captures also a sequence of screens in a movie file.

Controls

Snapshot

Format radio button Selects the format of the saved image file: PNG, JPEG or PDF.
Transparent background toggle Change the background to transparent when capturing a PNG snapshot.
Capture snapshot button Capture the viewer content and write it to a selected file in the given format.

Movie

Start/Stop recording button Capture a sequence of screenshots as a movie after selecting a file in WEBm, MP4 or MKV format.

Note: the recording starts as soon as the movie file has been selected and ends when recording is stopped.

Another note: AVI format is considered legacy so it is not provided. Use MP4 instead.

STL

Format radio button Selects the format of the saved STL file: ASCII or Binary.
Scale slider Scale the output geometry by this amount.
Capture geometry button Capture the current viewer content and write them to a selected STL format file.

Note: currently the STL file captures only atoms and bonds, not other geometries like polyhedra, unit cell or backbone tube.
