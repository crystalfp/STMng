Collection matcher

Match a structure against a collection of structures

This module matches crystal structures against a collection of structures. The match is done using the Oganov Valle fingerprints and ordering the result on increasing distance between the two fingerprints. To compute the fingerprints the atomic species are changed to hydrogen.

Controls

Enable collection matcher toggle Enable the matcher. Note that the process could be time consuming and computational heavy, so by default the module is disabled.
Neighbors slider Number of structures returned. The returned structures are ordered by increasing distance.
No max distance toggle If set do not limit the returned structure to distances less than the value set as “Max distance”.
Max distance numeric field Maximum value of distance of the returned structures from the test structure. This field is disabled if the “No max distance” toggle is set.
Similar structures clickable text List of the structures from the collection similar to the main one. Clicking on the entries opens the secondary window that shows the corresponding collection structure.
