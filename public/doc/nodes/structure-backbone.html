Structure backbone

Draw a backbone tube

Draw a backbone tube passing through given atoms and optionally pertains to a specific chain.

Controls

Show backbone toggle Made backbone one or more chains visible.
Threshold slider Threshold of one of the basis vectors to consider two adjacent backbone nodes on the same side of the unit cell. The slider is disabled if no unit cell is present in the structure.
Radius slider Radius of the backbone tube.
Select backbone atoms by radio buttons Select the backbone atoms by one of the following items.
Symbol
The atom type symbol.
Label
The atom label eventually present in the input structure. Note: for proteins the atoms are usually the ones with label "CA".
Index
Index of the atom inside the structure. The index starts from zero.
All
Select all atoms.
Backbone atoms selector text Space separated list of atoms selectors.
Select backbone segment list of toggles One toggle per backbone segment. This control is not present if the input structure has none or zero segments or chains defined.
Select All button Select all backbone segments.
Deselect All button Deselect all backbone segments.
