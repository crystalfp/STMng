Draw structure

Display the loaded structure

Render a structure to the screen.

Controls

Structure rendering mode radio buttons How to render the structure.
CPK
Atoms are represented with a radius proportional to their covalent radius.
VdW
Atoms are represented with a radius proportional to their Van der Waals radius. No bonds are rendered.
Licorice
Atoms are represented with a radius equal to the bond radius.
Lines
Bonds are rendered by lines. Atoms are not rendered.
Smooth color bonds toggle Bonds are colored by a smooth transition between the bonded atoms colors. Otherwise there are only two color bands. This does not apply to line rendering.
Show bonds strengths toggle In CPK mode modify the bonds diameter to reflect their strength as computed from the goodBonds values as used by USPEX.
Atom label radio buttons What represents the label on each atom.
Symbol
The atom type symbol.
Label
The atom label eventually present in the input structure.
Index
Index of the atom inside the structure. The index starts from zero.
Atom color radio buttons Selects color for atoms and bonds.
Type
The atoms are colored by the atom specie as defined in OpenBabel data.
Mono
Monochrome coloring. When selected, a color selector becomes visible below.
Bonds
Atoms are colored by their number of bonds. The color scale tries to use colors as different as possible.
Show legend toggle Display a legend for the coloring of bonds count. This control appears only when coloring atoms by bonds.
Visibility toggle buttons Selects what is rendered.
  • Structure
  • Bonds
  • Labels

The last button labelled "Hide" / "Show" deselect or set all previous buttons.
Visual parameters button Toggles the visibility of the next section containing parameters related to the visual appearance of the structure.

Visual parameters

Quality radio buttons Select the quality of the rendering by varying the number of subdivisions of spheres and cylinders.
  • Low
  • Medium
  • Good
  • Best
Bonds radius multiplier slider Multiplier for the standard bonds radius. It is enabled only for “CPK” and “Licorice” modes.
Sphere radius multiplier slider Multiplier for the standard spheres radius. It is enabled only for “CPK” mode.
Roughness slider Roughness of the surfaces.
Metalness slider Metal-like shining of the surfaces.
