X-Ray diffraction

Compute and display the X-Ray diffraction spectra

Compute and display the X-Ray diffraction spectra for the current structure. The code is ported from the one in the Pymatgen Python library.

Controls

Wavelength list Select the type of X-Ray source used to obtain the spectra. If selected the last entry of the list ("Other …") then a numeric field opens to manually enter the wavelength.
Numeric wavelength numeric field Manually enter the X-Ray source wavelength used to obtain the spectra. Appears only if the last entry of the list of wavelength codes ("Other …") is selected.
Two theta range range slider Angle range where the spectra should be computed.
Chart scaled toggle When set, the intensities are scaled to have 100 as the maximum value.
Show HKL toggle When set, near the peaks on the chart show the corresponding HKL values with the relative multiplicity.
Peak width slider Width of the peaks at half maximum (FWHM). If set to zero show the peaks without smoothing.
Open chart button Open a window showing the diffraction spectra and the hkl plus multiplicity peak values. The button is disabled if no structure has been loaded. Note that hovering the mouse on the chart show the corresponding spectra value. This window has also a button to create a chart snapshot as a PNG image file selecting if a solid or transparent background is requested.
