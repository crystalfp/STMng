X-Ray diffraction

Compute and display the X-Ray diffraction spectra

Compute and display the X-Ray diffraction spectra for the current structure. The code is ported from the one in the Pymatgen Python library.

From the atomic positions, STMng solves Braggs Law to identify peak positions and diffraction angle. Peak intensities are obtained from the structure factor. Combining the Miller indices, the angle of diffraction and structure factor, it plots the powder diffraction pattern.

The spectra maximum peak intensity is normalized to 100.

Controls

Wavelength list Select the type of X-Ray source used to obtain the spectra. If the last entry of the list ("Manual") is selected, then a numeric field opens to manually enter the wavelength.
Numeric wavelength numeric field Manually enter the X-Ray source wavelength used to obtain the spectra. Appears only if the last entry of the list of wavelength codes ("Manual") is selected.
Two theta range range slider Range of angles where the spectra should be computed.
Show HKL toggle When set, near the peaks on the chart display the corresponding hkl values with the relative multiplicity. Note that, to have the points coincident to the spectra peaks, the peak width should be zero.
Peak width slider Width of the peaks at half maximum (FWHM). If set to zero show the peaks without smoothing.
Peak threshold slider Computed peaks with normalized intensities below this threshold are ignored.
Open chart button Open a window showing the diffraction spectra and the hkl plus multiplicity peak values. The button is disabled if no structure has been loaded or if the structure has no unit cell.
