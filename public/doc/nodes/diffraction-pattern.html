X-Ray diffraction

Compute and display the X-Ray diffraction spectra

Compute and display the X-Ray diffraction spectra for the current structure. The code is bases on the one from the pymatgen Python library.

Controls

Wavelength list Select the type of X-Ray source used to obtain the spectra.
Two theta range range slider Angle range where the spectra should be computed.
Chart scaled toggle When set, the intensities are scaled to have 100 as the maximum value.
Peak width slider Width of the peaks at half maximum (FWHM).
Open chart button Open a window showing the diffraction spectra. The button is disabled if no structure has been loaded. Note that hovering the mouse on the chart show the corresponding spectra value. The chart has also a button to create a chart snapshot as a PNG image file.
