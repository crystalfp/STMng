Isosurface

Isosurfaces from volumetric data

Draw surfaces that touch points with a given value in the volume.

Controls

Show isosurface toggle Made isosurface visible.
Dataset slider Which volumetric data set should be used.
Nested isosurfaces toggle Draw more than one isosurface. The following three controls appear if this is set.
Number of isosurfaces slider Number of nested isosurfaces to create.
Values range range slider The volumetric data values range. Sets values of the first and last isosurfaces.
Limit colormap to range toggle Only values in the selected range are mapped to the colorscale, otherwise the full value range is mapped.
Isosurface value slider Value to which the isosurface should be draw. This control appears only if “Nested isosurfaces” is not set.
Colormap selector Name of the colormap to convert values into colors.
Opacity slider Opacity of the isosurfaces.
