Isosurface

Isosurfaces from volumetric data

Draw surfaces that touch points with a given value in the volume.

Controls

Show isosurface toggle Made isosurface visible.
Dataset slider Which volumetric data set should be used.
Nested isosurfaces button Draw more than one isosurface. The following two controls appear if this is set.
Number of isosurfaces slider Number of nested isosurfaces to create.
Values range range slider The volumetric data values range. Sets values of the first and last isosurfaces.
Isosurface value slider Value to which the isosurface should be draw. This control appears only if no nested isosurfaces is selected.
Colormap selector Name of the colormap to convert value into color.
Opacity slider Opacity of the isosurfaces.
