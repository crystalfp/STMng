Fingerprints

Compute fingerprints for a set of structures to group them

The module implements the fingerprinting method to group similar structures as presented in:

  • Valle, M., & Oganov, A. R. (2008). Crystal Structures Classifier for an Evolutionary Algorithm Structure Predictor. IEEE Symposium On Visual Analytics Science and Technology, VAST 2008, 11–18
  • Valle, M., & Oganov, A. R. (2010). Crystal fingerprint space – a novel paradigm for studying crystal-structure sets. Acta Crystallographica Section A, 66(5), 507–517

Controls

The controls are grouped in logical steps that normally should be completed top to bottom.

Energy from minimum / Max energy numeric field For filtering structures enter the maximum energy the structures should have or, when “Threshold from minimum energy” is set, the offset from the minimum energy of the accumulated structures to establish the threshold.

Structures selected readonly text Count of structures eventually selected by energy over the total number of structures.

Compute fingerprints

Structures are nanoclusters toggle Mark the structures as nanoclusters, that is, they are not crystals with a periodic structure.
Force cutoff at: toggle + numeric field When set, forces the cutoff distance for the computation of fingerprint to the value entered.
Cutoff readonly text The cutoff distance (computed or manually entered) that will be used by the fingerprint computation.
Fingerprinting method list Select the method for computing the fingerprints:
Oganov-Valle fingerprint
A rapidly convergent function based on distance distributions and related to RDF and diffraction spectra. It is the first method proposed in the original paper. Needs bin size and peak width. Not for nanoclusters.
Chemical scale weighted fingerprint
It is the same as “Oganov-Valle fingerprint”, but the atom Z values are mapped to the Chemical Scale correction for a given atom type.
Per element fingerprint
The fingerprint is composed by a section for each atom type in the unit cell. Each section is an ascending ordered set of distances from the corresponding atom to all the other atoms of the extended unit cell. All these sets contain the same number of distances. Furthermore each section is labeled with the element type of the corresponding atom. Needs bin size and peak width.
Per site fingerprint
It is the same as “Per element fingerprint” but the centroid of all fingerprints is removed from each fingerprint. Needs bin size and peak width.
Peak width numeric field The distance peaks are smoothed with a gaussian and then binned.
Bin size numeric field To convert smoothed distance peaks into a vector of values, that is, the fingerprint, the smoothed interatomic distances are binned to this size.
Compute fingerprints button Compute the fingerprints for the selected structures.
Dimensionality readonly text At the end of the computation shows the length of the computed fingerprints.

Compute distances

Distance method list Select the measuring method to obtain the distance between fingerprints:
Cosine distance
Cosine distance between fingerprints interpreted as multidimensional vectors. The distances are scaled between 0 and 1. This measure violates the triangle inequality.
Euclidean distance
Euclidean distance between fingerprints interpreted as points in a multidimensional space.
Minkowski distance of order ⅓
The Minkowski distance with order p = ⅓. With this order value, the measure violates the triangle inequality but spread points better in the multidimensional space facilitating grouping.
Fix triangle inequality toggle Iteratively adjust distances to satisfy the triangle inequality for every triples of distances.
Compute distances button Compute the distances between fingerprints.
Distances computed readonly text At the end of the computation shows the number of computed distances.

Structure landscape

Grouping method list Select the method to group similar fingerprints:
Pseudo SNN
Structure clustering method based on the number of nearest neighbors points shared between pairs of connected points and confirm this last connection only if the number is at least K (equal to one plus the margin value). SNN means Shared Nearest Neighbor and is “Pseudo” becouse laks the final DBSCAN step to refine clustering.
Hierarchical grouping (single linkage)
It is a methods of hierarchical clustering based on grouping clusters in bottom-up fashion, at each step combining two clusters that contain the closest pair of elements not yet belonging to the same cluster as each other.
Hierarchical grouping (complete linkage)
It is an agglomerative hierarchical clustering that at the beginning of the process, has each element is in a cluster of its own. The clusters are then sequentially combined into larger clusters until all elements end up being in the same cluster.
Distance thresh. % numeric field Distance threshold for two structures to consider them in the same group. The distance is given as a percentage of the global maximum distance.
Margin numeric field For the Pseudo SNN method it is the number of connection above one to consider two structures connected.
Classify structures button Classify fingerprints into groups.
Groups found readonly text At the end of the computation shows the number of groups found.

Fingerprints scatterplot window

The scatterplot window shows the fingerprints as points in a 2D space. The points are colored by the associated value.

This window shows five different kinds of scatterplot:

Group
One point for each structure colored by group.
Energy
One point for each structure colored by energy.
Fidelity
One point for each distance with X value the distance and Y value the projected distance. More the points live near the diagonal, more the projection preserves the original distance. Note that the colorscale goes from white for distances that have a perfect mapping, to black for bad mapped distances. Note also that, for performance reasons, the number of points is reduced to less than 40,000.
Quality
Grouping quality using the silhouette coefficients. These coefficients goes from 1 that show perfect association of the point to the group, to -1 that show that the point is in the wrong group. There are two special cases: the point is alone in a group and the point is on the border between two groups.
From min
One point for each structure colored by its distance from the point with minimum energy.

The grouping quality color scale is:

  • ⬚ Strong (silhouette > 0.7)
  • ⬚ Reasonable (silhouette > 0.5)
  • ⬚ Weak (silhouette > 0.25)
  • ⬚ Bad (silhouette > 0.05)
  • ⬚ Very bad (silhouette < -0.05)
  • ⬚ Cannot measure (group of one element)
  • ⬚ On the border between two groups (silhouette between -0.05 and 0.05)

Clicking on one point selects it and show a small popup.

Dragging the mouse with the right button down makes a rectangular selection and select all points inside. If done with the control key pressed, deselects all selected points inside the selection.

Note that due to the random initialization of the projection method, sometime the direction of the axis flips.

Besides the scatterplot kind selector, the scatterplot window has the following controls:

Point radius slider Radius of the scatterplot points.
Deselect button Deselect all selected points.
Show legend toggle Show a legend for the current type of plot.
Close button Close the window.

The left side of the window contains controls to manage point selection.

Select all button Select all points.
Deselect all button Deselect all selected points.
Group slider Select one of the groups present.
Select group button Select all points pertaining to the selected group.
Min energy per group button For each group select the point with minimum energy within the group.
Generalized convex hull button Compute the generalized convex hull of the points projected to 3D space plus corresponding energy. From this surface selects only the vertices on the lowest side of the hull. Setting a threshold > 0 remove similar points that dist less than this value. For a group of similar points extracts the lowest energy one.
Threshold slider Set the maximum distance to consider two points from the convex hull similar. If zero, no similar points removal is done.
Compare selected button Visually compare pairs of structures. Not yet implemented.
Export selected button Export the structures whose points have been selected to a POSCAR file and a corresponding energy file.

Fingerprints charts window

This window shows two different kinds of charts:

Fingerprint
Fingerprint for the selected structure step.
Energy-Dist
Chart of energy differences from the minimum energy versus the distance from the point of minimal energy.

Besides the chart kind selector, the window has the following controls:

Structure step selector slider Select the structure step for which the fingerprint should be charted. This control appears only when fingerprint chart is selected.
Close button Close the window.

Energy landscape window

Show the energy surface interpolating the computed fingerprints and the corresponding energies.

Scale slider Vertical scale of the plot.
Grid side slider Side of the resulting square surface. It is a power of two.
Power slider Parameter for the interpolation. Normally it is two, bigger values make the points influence be more local to a given point.
Colormap selector Name of the colormap to convert value into color.
Close button Close the window.
