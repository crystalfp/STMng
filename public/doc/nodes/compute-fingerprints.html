Fingerprints

Compute fingerprints for a set of structures to group them

The module implements the fingerprinting method to group similar structures as presented in:

  • Valle, M., & Oganov, A. R. (2008). Crystal Structures Classifier for an Evolutionary Algorithm Structure Predictor. IEEE Symposium On Visual Analytics Science and Technology, VAST 2008, 11–18
  • Valle, M., & Oganov, A. R. (2010). Crystal fingerprint space – a novel paradigm for studying crystal-structure sets. Acta Crystallographica Section A, 66(5), 507–517

Note that this module is still work in progress.

Controls

The controls are grouped in logical steps that normally should be completed top to bottom.

Accumulate structures

Start/stop accumulating button Toggle accumulating structures for which fingerprint should be computed. This toggle is present also in the reader page, when this module is loaded.
Reset button Clear the accumulated structures.
Structures are nanoclusters toggle Mark the structures as nanoclusters, not crystals with a periodic structure.
Structures loaded readonly text Count of accumulated structures.

Filter structures

Select energy file file selector Select the file with energies per structure. The file is ASCII with one energy per line. The energies will be associated to the accumulated structures and will be used also to filter the structures.
Filter by energy toggle From the accumulated structures extract only the ones that satisfy the following constrains.
Threshold from minimum energy toggle The maximum energy threshold is computed from the minimum energy of accumulated structures.
Energy from minimum / Max energy numeric field For filtering structures enter the maximum energy the structures should have or, when "Threshold from minimum energy" is set, the offset from the minimum energy of the accumulated structures to establish the threshold.
Max energy readonly numeric field The computed or manually entered energy threshold.
Structures selected readonly text Count of structures eventually selected by energy.

Compute fingerprints

Force cutoff at: toggle + numeric field When set, forces the cutoff distance for the computation of fingerprint to the value entered.
Cutoff readonly text The cutoff distance (computed or manually entered) that will be used by the fingerprint computation.
Fingerprinting method list Select the method for computing the fingerprints.
Normalized diffraction
Needs bin size and peak width. Not for nanoclusters.
Mendeleev spectra
Needs bin size and peak width. Not for nanoclusters.
Chemical scale spectra
Needs bin size and peak width. Not for nanoclusters.
Per element diffraction
Needs bin size and peak width.
Distances per atom
Not for nanoclusters.
Merged distances
Not for nanoclusters.
Re-centered per element diffraction
Needs bin size and peak width.
Trimmed per element diffraction
Bin size numeric field To reduce the dimension of the fingerprints, the interatomic distances are binned to this size.
Peak width numeric field The binned distances are smoothed and so cover more than one bin.
Compute fingerprints button Compute the fingerprints for the selected structures.
Dimensionality readonly text At the end of the computation shows the length of the computed fingerprints.

Compute distances

Distance method list Select the method to measure the distance between fingerprints.
Cosine distance
Cosine distance between fingerprints interpreted as multidimensional vectors. The distances are scaled between 0 and 1
Euclidean distance
Euclidean distance between fingerprints interpreted as points in a multidimensional space
Minkowski distance of order ⅓
The Minkowski distance with order p = ⅓. With this order value the measure violates the triangle inequality
Fix triangle inequality toggle Iteratively adjust distances to satisfy the triangle inequality for every triple of distances.
Compute distances button Compute the distances between fingerprints.
Distances computed readonly text At the end of the computation shows the number of computed distances.

Classify structures

Grouping method list Select the method to group similar fingerprints.
Pseudo SNN
Use the margin value
Hierarchical grouping (single linkage)
Hierarchical grouping (complete linkage)
Distance thresh. % numeric field The percentage of the maximum distance to consider two structures in the same group.
Margin value (K) numeric field For the method that require it is the number of connection above one to consider two structures connected.
Classify structures button Classify fingerprints into groups.
Groups found readonly text At the end of the computation shows the number of groups found.

Show results

Show scatterplot button Open the Fingerprints scatterplot window

Fingerprints scatterplot window

The scatterplot window shows the fingerprints in a 2D space. The fingerprints are colored by the group they belong to or by the energy.

This window shows three different kinds of scatterplot:

Group
One point for each structure colored by group
Energy
One point for each structure colored by energy
Efficiency
On point for each distance with X value the distance and Y value the projected distance. More the points live near the diagonal, more the projection preserves the original distance

Clicking on one point selects it and show a small popup. This popup possibly does not stay in the final version.

Besides the scatterplot kind selector, the scatterplot window has the following controls:

Point radius slider Radius of the scatterplot points
Deselect button Deselect all selected points
Close button Close the window
