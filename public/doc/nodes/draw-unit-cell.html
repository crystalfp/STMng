Cell

Visualize the unit cell and the supercell

Draw the unit cell and generates and draw supercell.

The input structure should have basis vectors defined.

The node outputs the input structure replicated if the repetitions along any basis is bigger than 1.

Controls

Show unit cell toggle Show the unit cell borders.
Dashed lines toggle Show the unit cell using dashed lines. Disabled if the line width is not zero.
Show basis vectors toggle Show the basis vectors as arrows.
Line width slider Width of the border lines. Zero is the minimum width. Over zero the lines are rendered as small tubes.
Line color color selector Select the color of the unit cell lines.
Cell repetitions sliders (3) Repetition of the unit cell along each basis vector.
Reset Button Reset the cell repetitions to 1.
Show supercell toggle Show the supercell borders.
Dashed lines supercell toggle Show the supercell using dashed lines. Disabled if the line width is not zero.
Line color color selector Select the color of the supercell lines.
Shift origin sliders (3) Shift the unit cell along basis vectors by a percentage of them by changing the cell origin.
Shrink cell toggle Instead of shifting the unit cell, shrink it.
Reset Button Reset the cell shift to zero and shrink cell to false.
