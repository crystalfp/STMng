Prototype matcher

Match a structure against known “prototype” structures

This module matches crystal structures against known crystallographic “prototype” structures. The code has been ported from the one in the Pymatgen Python library.

The prototype found is shown by its “mineral” tag field that, in the Aflow’s Encyclopedia of crystallographic prototypes, is identified as the “Notes” field. These values have been updated by inputs provided by Prof. Artem Oganov. If you find that any string could be made clearer or more precise, do not hesitate to contact the author.

Controls

Enable prototype matcher toggle Enable the matcher. Note that the process could be time consuming and computational heavy, so by default the module is disabled.
Fractional length tolerance slider Tolerance for the unit cell sides during structure match.
Site tolerance slider Tolerance for the fractional atoms’ positions during structure match.
Angle tolerance slider Tolerance for the lattice angles for the structure matcher.
Reset parameters button Reset parameters to the default values.
Prototypes for formula readonly text Show the empirical formula for the loaded compound.
Matched structure readonly text List of the “mineral” tags from the matched prototypes and the corresponding “Aflow UID” string. If no match is found “No match found” is shown. Note that the tag field could be overridden by an internal map from the “Aflow UID” to a different one.

Credits

This module uses data from the AFLOW LIBRARY OF CRYSTALLOGRAPHIC PROTOTYPES described in: Mehl, M. J., Hicks, D., Toher, C., Levy, O., Hanson, R. M., Hart, G., & Curtarolo, S. (2017). The AFLOW library of crystallographic prototypes: part 1. Computational Materials Science, 136, S1-S828. DOI: 10.1016/j.commatsci.2017.01.017

The code has been ported from the Pymatgen library’s AflowPrototypeMatcher class.

The Aflow library has made the Encyclopedia of crystallographic prototypes available which can be searched by the retrieved UID string.
