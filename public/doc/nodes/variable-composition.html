Variable composition

Split compositions and remove duplicates structures

The module splits input structures by their composition. Then, for each composition, computes the structure fingerprints and their distances to remove duplicates. At the end could show a chart of the convex hull results and could save the reduced compositions in different files or consolidate them in a single one.

Controls

Accumulated structures

Count readonly text Count of accumulated structures. The start and stop of the accumulation are done from the Structure Reader page.
Reset button Clear the accumulated structures.
Number of components radio buttons Number of component in the input structures. The count values could range from 2 to 4.
End-member compositions editable table Each row is a component in the input structures. There is an atomic specie per column ordered alphabetically and the numeric values is the multiplicity of this specie in the given component. For example for MgO + SiO2 the two rows should be 1, 1, 0 and 0, 2, 1.
Compute compositions button Compute the proportion of the given components for each structure accumulated. If for some structure the components are not valid, an error message appears. Note that for some compositions, where there is ambiguity, it reports that it cannot proceed.

Compositions

The data table has the following columns:

SelectSelect for the analysis and output the given composition. All compositions are selected by default. The toggle in the header selects or deselects all compositions.
CompositionThe proportion of the various components.
CountThe number of structures with the given composition. The column is sortable by clicking on the column header.
RemainAfter filtering, computing distances and removing duplicates, this column is populated with the resulting number of structures. The column is sortable by clicking on the column header.

Filter structures

Filter toggle When set, remove from input the structures with a distance from convex hull greater than “Convex hull dist.”.
Convex hull dist. numeric field The maximum distance of a structure from the convex hull to be retained. The field is disabled if filter toggle is not set.
Remaining readonly numeric field The number of structures remaining after filtering.

Compute distances

Force cutoff at: toggle + numeric field When set, forces the cutoff distance for the computation of fingerprint to the value entered.
Fingerprinting method list Select the method for computing the fingerprints needed to compute distances and remove duplicated structures. The methods are listed in the Compute fingerprints node documentation.
Peak width numeric field The distance values are smoothed with a gaussian of this full width at half maximum (FWHM) value and then binned.
Bin size numeric field To convert smoothed distance peaks into a vector of values, that is, the fingerprint, the smoothed interatomic distances are binned to this size.
Distance method list Select the measuring method to obtain the distance between fingerprints. The methods are listed in the Compute fingerprints node documentation.
Fix triangle inequality toggle Iteratively adjust distances to satisfy the triangle inequality for every triples of distances.

Remove duplicates

Remove toggle Enable removal of duplicated structures, that is, structures with a distance between fingerprints less than the distance threshold.
Distance threshold numeric field The maximum distance between two fingerprints to consider the further one from the convex hull for removal. If the energies are missing, retains the most central point in the group of near fingerprints.
Remaining n of m readonly field When the “Analyze selected for duplicates” is running this field show the number of duplicates removed over the analyzed structures so far.
Analyze selected for duplicates button Start the computation of the fingerprints and distances and the removal of duplicated structures. The compositions are computed in increasing count of structures. The computations are not parallelized but done one by one.
Show chart button Open a window to show the convex hull of the compositions. This operation is enabled for a number of components of 2 or 3.
3D view button Open a window to show the convex hull in 3D of the compositions. This operation is enabled for a number of components of 3 or 4.
Consolidate output toggle When set, “Save analyzed” saves all resulting structures into a single file ordered by energy and then by distance from the convex hull. Otherwise there is a file per composition with structures ordered by energy and then by increasing distance from the convex hull.
Save analyzed button Show the output directory selector to save the analysis output.
Output directory directory selector Select the directory where the output files will be saved. There is a file (or a pair of files) for each selected composition. The files are called composition‑<proportions>.poscar. If they have an associated per atom energy, the file composition‑<proportions>.energy is created with them. If the “Consolidate output” toggle is set, the proportions part of the filenames is changed to the “all” string.
Files saved readonly text At the end of the save step, show the number of files created.
