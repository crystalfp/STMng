Compute crystal shape

Display the crystal shape based on the Wulff theorem

Given a crystal structure, this node finds its shape by intersecting the hkl planes according to the Wulff theorem.

Controls

Use all HKL planes toggle When set use all the planes found in the first step. Note that this could generate an out of memory error.
Max HKL planes to use slider When the previous toggle is not set, this value limits how many planes to use. Normally 100 is sufficient, but if there are memory problems could be lowered.
Multi process parallelism toggle Select to parallelize computation using multiple processes. If not set the parallelization is done using threads in a single process. The multi-process method is slower, but gives more memory to each computation. Select it if selecting parallelization on threads has memory problems.
Compute crystal shape button Start the crystal shape computation and then open the Crystal shape window to display the result. If nothing ha changed, this button does not recompute the shape, only displays it.
Current step readonly text During computation, this field shows the current step of the computation.
