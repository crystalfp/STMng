Compute crystal shape

Display the crystal shape based on the Wulff theorem

Given a structure this node find the shape by intersecting the hkl planes according to the Wulff theorem.

Controls

Use all HKL planes toggle When set use all the planes found in the first step. Note that this could generate an out of memory error.
Max HKL planes to use slider When not using all planes, this limits how many planes to use. Normally 100 is sufficient, but if there are memory problems could be lowered.
Compute crystal shape button Start the crystal shape computation and then open the Crystal shape window to display the result. If nothing ha changed, this button does not recompute the shape, only display it.
Current step readonly text During computation, this field shows the current step of the computation.
