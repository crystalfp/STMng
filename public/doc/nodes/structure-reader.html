Structure Reader

Description

Read structural chemical data. To specify a trajectory the file could contain a single structure (single frame) or a sequence of structures (Multiframe from file)

Parameters

Select file format list Select the format of the file to be read. The available formats.
Atoms types input (space separated list) Enter the symbols of the atoms present in the file. This field appears only for POSCAR, CHGCAR, LAMMPS and LAMMPStrj formats.
Load file selector Select the file to be read.

The following fields appears if the format contains multiple steps:

Loop switch Play the steps in a loop.
Movie from steps switch Start recording a movie as soon as the steps animation starts. As soon as the animation stop, a file selector will ask for the movie file to be created.
Step slider Move between the steps and show the actual step.
Control buttons buttons Standard control buttons for the steps playback. From left: go to beginning, go to previous step, play/stop, co to next step, go to last step.
