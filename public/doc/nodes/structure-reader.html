Reader

Structure reader

Read structural chemical data. The file could contain a single structure or a sequence of structures to specify a trajectory or a set of USPEX results.

If the format needs an auxiliary file, the module shows its file selector.

The files could be also drag from elsewhere and dropped on the file selectors.

There are also two special formats: “Prototypes”, that queries and load one prototype from the Aflow prototype database, and “Collection”, that queries and load one structure from a collection of useful structures.

If set in the Viewer 3D controls, the camera position could be reset at each frame loaded.

Controls

File format list Select the format of the file to be read from the list of available formats. At the end of the list a section titled “Special sources” contains sources that are not coming from a file. These are:
  • Prototypes: opens the prototypes search and load panel (see below) instead of the following controls.
  • Collection: open a search panel to search files in the structure collection (see below) instead of the following controls.
Atoms types input (space separated list) Enter the symbols of the atoms present in the file. This field appears only for POSCAR, POSCAR + XDATCAR, POSCAR + ENERGY, CHGCAR, LAMMPS and LAMMPStrj formats. The field content is accepted only with the ENTER key or moving the focus away from the field. The atoms types could be changed after file loading.
Append toggle If set, the next file read will have its content appended to what has already been read. Note that for the POSCAR + XDATCAR format the POSCAR file is read once to give the unit cell parameters even if the “Append” toggle is set.
Select or drop input file file selector Select or drop here the file to be read.
File has energy per atom toggle For POSCAR + ENERGY format selects if the energy file has energy per atom instead of energy per structure. Note that this toggle should be set before loading the ENERGY file.
Select or drop <auxiliary format> file file selector Select or drop here the auxiliary file associated to the input file, to be read. Show the auxiliary file format name. This field appears only if needed.
Use Bohr units toggle For the Gaussian cube format selects the measurement unit to be used: Bohr or Å.
Read hydrogens toggle For the PDB format selects if hydrogen atoms should be loaded and visualized.

The following fields appear if the format contains multiple steps:

Movie from steps toggle This toggle appears only if the Capture module is present in the project. Start recording a movie as soon as the steps animation starts. As soon as the animation stop, a file selector will ask for the movie file to be created.
Record trajectories toggle This toggle appears only if the Trajectories module is present in the project. Starts recording trajectories. It is disabled if no atom to trace has been selected in the Trajectories module.
Accumulate for fingerprinting toggle This toggle appears only if the Fingerprinting module is present in the project. It enables accumulating structures for fingerprinting.
Collect for structure writer toggle This toggle appears only if the Writer module is present in the project. Starts accumulating structures to be saved in a single file.
Collect for variable composition toggle This toggle appears only if the Variable composition module is present in the project. It enables accumulating structures for decomposing them into components.
Loop toggle Play the steps in a loop.
Reverse toggle Play the steps in reverse direction.
Step increment numeric spinner Set how many steps the system does at every tick.
Step slider Move between the steps and show the actual step and the total number of steps. If speed is set to “Fast” this slider is not visible.
Step range range slider Set the first and last steps to be visualized. When the first step value is more than one or the final value is less than the number of steps from the file, a Range label became visible with the first and last steps and the button to reset its values to cover the full range of steps from file. Note that this slider is not labelled.
Control buttons buttons Standard control buttons for the steps playback. From left: go to beginning, go to previous step, play/stop, go to next step, go to last step.
Speed radio buttons Speed of playback by defining a delay between steps.
Fast
No delay between steps. Note: to speedup playback, the step slider is hidden.
Medium
200 msec delay between steps.
Slow
400 msec delay between steps.

The following fields appear if the selected format is “Prototypes”:

Prototype query text with autocomplete Query the full list of prototypes. The query show a list of possible minerals/aflow UID and each character entered shorten the list.
Query result readonly text When a prototype has been selected, here appears its name (called “mineral” in aflow database), the aflow UID, its strukturbericht and the Pearson values.

The following field appears if the selected format is “Collection”:

Collection query text with autocomplete Query the full list of structures in the collection. The query show a list of possible structure file names and each character entered shorten the list.

Credits

The Aflow project has made available the Encyclopedia of crystallographic prototypes which can be searched and loaded by part of the Aflow UID string or mineral name.

The prototypes database is described in: Mehl, M. J., Hicks, D., Toher, C., Levy, O., Hanson, R. M., Hart, G., & Curtarolo, S. (2017). The AFLOW library of crystallographic prototypes: part 1. Computational Materials Science, 136, S1-S828. DOI: 10.1016/j.commatsci.2017.01.017

The files in the Collection have been collected from various sources and provided by Prof. Artem Oganov.
