Structure Reader

Read structural chemical data. To specify a trajectory the file could contain a single structure (single frame) or a sequence of structures (Multiframe from file)

If the format needs an auxiliary file, the module adds this file selector.

Controls

File format list Select the format of the file to be read. The list of available formats.
Atoms types input (space separated list) Enter the symbols of the atoms present in the file. This field appears only for POSCAR, POSCAR + XDATCAR, CHGCAR, LAMMPS and LAMMPStrj formats. The field content is accepted only with ENTER or moving the focus away from the field. The atoms types could be changed after file loading.
Select input file file selector Select the file to be read.
Select <auxiliary format> file file selector Select the auxiliary file to be read. Show the auxiliary file format name. Appears only if needed.
Use Bohr units toggle For Gaussian cube format selects the measurement unit to be used: Bohr or Å.

The following fields appear if the format contains multiple steps:

Loop toggle Play the steps in a loop.
Movie from steps toggle This switch appears only if the Capture module is present. Start recording a movie as soon as the steps animation starts. As soon as the animation stop, a file selector will ask for the movie file to be created.
Record trajectories toggle This switch appears only if the Trajectories module is present. Starts recording trajectories.
Accumulate for fingerprinting toggle This switch appears only if the Fingerprinting module is present. Starts accumulating structures for fingerprinting.
Collect for structure writer toggle This switch appears only if the Writer module is present. Starts accumulating structures to be saved in a single file.
Step slider Move between the steps and show the actual step and the total number of steps.
Control buttons buttons Standard control buttons for the steps playback. From left: go to beginning, go to previous step, play/stop, go to next step, go to last step.
