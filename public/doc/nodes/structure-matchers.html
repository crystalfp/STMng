Structure matchers

Match a structure against known “Prototype” and “Collection” structures

This module matches crystal structures against two collections of structures: known crystallographic “prototypes” structures from the Encyclopedia of crystallographic prototypes and a collection of interesting structures from various sources.

The “Prototype” structures are accessible as a Structure reader special collection. The match is done using code ported from the Pymatgen Python library class AflowPrototypeMatcher. The prototype found is shown by its “mineral” tag field that, in the Aflow’s Encyclopedia of crystallographic prototypes, is identified as the “Notes” field. These values have been updated by inputs provided by Prof. Artem Oganov. If you find that any string could be made clearer or more precise, do not hesitate to contact the author.

The “Collection” structures are accessible as a Structure reader special collection. The match is done using the Oganov Valle fingerprints (see the Compute fingerprints node) and ordering the result on increasing cosine distance between the two fingerprints. To compute the fingerprints all the atomic species are changed to hydrogen.

Controls

Common controls

Enable matchers toggle Enable the matchers. Note that the process could be time consuming and computational heavy, so by default the module is disabled.
Input structure: formula readonly text Show the empirical formula for the loaded structure.
Reset parameters button Reset parameters to the default values.

Collection matcher

Neighbors slider Number of structures returned. The returned structures are ordered by increasing cosine distance.
No max distance toggle If set the returned structures’ distances are not limited to be less than the value set as “Max distance”.
Max distance numeric field Maximum value of distance of the returned structures from the test structure. This field is disabled if the “No max distance” toggle is set. Note that cosine distances goes from 0 to 1.
Similar structures clickable text List of the structures from the collection similar to the input one and their distances. Clicking on the entries opens the secondary window that shows the corresponding collection structure.

Prototype matcher

Fractional length tolerance slider Tolerance for the unit cell sides during structure matching.
Site tolerance slider Tolerance for the fractional atoms’ positions during structure matching.
Angle tolerance slider Tolerance for the lattice angles for the structure matcher.
Matched structure clickable text List of the “mineral” tags from the matched prototypes and the corresponding “Aflow UID” string. If no match is found “No match found” is shown. Note that the tag field could be overridden by an internal map from the “Aflow UID” to a different one. Clicking on the entries opens the secondary window that shows the corresponding prototype structure.

Credits

The Aflow project has made available the Encyclopedia of crystallographic prototypes which can be searched and loaded by part of the Aflow UID string or mineral name.

The prototypes database is described in: Mehl, M. J., Hicks, D., Toher, C., Levy, O., Hanson, R. M., Hart, G., & Curtarolo, S. (2017). The AFLOW library of crystallographic prototypes: part 1. Computational Materials Science, 136, S1-S828. DOI: 10.1016/j.commatsci.2017.01.017

The files in the Collection have been collected from various sources and provided by Prof. Artem Oganov.
