Interpolate

Interpolate volume data

Interpolate a given scalar volume set adding points between original grid points. The interpolation is done using a tricubic polynomial.

The input structure should have volumetric data in a grid at least 4x4x4, otherwise it is simply copied to output.

The volumetric data sets not selected are simply copied to output.

Controls

Interpolate volume data toggle Enable the interpolation, otherwise the input structure is simply copied to output.
Dataset slider Which volumetric data set should be interpolated.
Points to add slider Points (1–10) to be added between original grid points.
