Orthoslice

Orthoslice of volumetric data

Draw an isosurface, a plane perpendicular to a chosen axis and map the values on it.

It draw isolines on these planes if requested.

Controls

Show orthoslice toggle Make orthoslice visible.
Dataset slider Which volumetric data set should be used.
Axis radio buttons Perpendicular to which axis the slice is taken.
Plane slider Which plane of the volumetric data should be taken.
Values range range slider The volumetric data values range. Could be made smaller than the full values range to highlight a specific set of values.
Use discrete classes toggle Divide the range in bands instead of a continuum range of colors between minimum and maximum.
Number of classes slider Number of bands to create. This control appears only if “Use discrete classes” is selected.
Colormap selector Name of the colormap to convert values into colors.
Show isolines toggle Show isolines (one if not using discrete classes, otherwise multiple).
Color isolines toggle Color isolines by the value they take.
Isoline value slider When not using multiple classes, set the value of the single isoline.

Credits

Isolines computation is based on the marching square algorithm.
