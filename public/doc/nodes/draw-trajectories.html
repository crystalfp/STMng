Trajectories

Draw trajectories of atoms

Draw selected atoms trajectories as lines or cloud avoiding large jumps. Line and clouds are colored by the atom color they trace.

When the Mean displacement visualizer window is visible, the module outputs the mean structure and mean unit cell (if present) as another structure to be visualized alongside the original structure. Note that this arrangement is not present in the default project for performance reasons, but is possible to setup it simply using the project editor.

Controls

Select traced atoms by radio buttons Select the traced atoms by one of the following methods:
Symbol
The atom type symbol.
Label
The atom label eventually present in the input structure.
Index
Index of the atom inside the structure. The index starts from zero.
All
Select all atoms.

Note: if the tracing is running, this control is deactivated. This happens also if the run is selected on the Reader page.
Traced atoms selector text Space separated list of atoms selectors.
Show trajectories toggle Made trajectories visible.
Max displacement slider Maximum displacement between two atoms positions to consider them a single trace. Visible only when “Show trajectories” is selected.
Show position clouds toggle Draw a cloud of positions visited by the atoms.
Point sprite size slider Size of the sprite texture that will be accumulated on each point as percent of the normal size.
Show averages button Open the Mean displacement visualizer window that lists the mean position and mean displacement of the selected atoms.
Start/stop trajectories button Start or stop accumulating trajectories.
Clear trajectories button Clear the accumulated trajectories.
