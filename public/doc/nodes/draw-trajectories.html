Trajectories

Draw trajectories of atoms

⚠ Work in progress! ⚠

Controls

Show trajectories toggle Made trajectories visible.
Clear trajectories button Clear the accumulated atoms positions.
Select traced atoms by radio buttons Select the traced atoms by one of the following items.
Symbol
The atom type symbol.
Label
The atom label eventually present in the input structure.
Index
Index of the atom inside the structure. The index starts from zero.
All
Select all atoms.
Traced atoms selector text Space separated list of atoms selectors.
Max displacement slider Maximum displacement between two atoms positions to consider them a single trace.
Show position clouds toggle Draw cloud of positions visited by the atoms. It is still experimental.
Cloud volume subdivisions slider Side subdivisions of the volume containing the position clouds.
Enlarge volume slider How much the cloud volume is enlarged to contain the whole cell.
Start trajectories button Start accumulating trajectories.
