Polyhedra

Draw polyhedra around selected atoms

Draw polyhedra passing through bonded atoms to selected atoms.

Controls

Show polyhedra toggle Made polyhedra visible.
Color by center atom toggle Instead to have polyhedra white colored, colors them by central atom color.
Select central atoms by radio buttons Select the central atoms by one of the following items.
Symbol
The atom type symbol.
Label
The atom label eventually present in the input structure.
Index
Index of the atom inside the structure. The index starts from zero.
All
Select all atoms.
Central atoms selector text Space separated list of atoms selectors.
Opacity slider Opacity of the polyhedra facies.
