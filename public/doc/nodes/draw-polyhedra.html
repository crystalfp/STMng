Polyhedra

Draw polyhedra and triangles around selected atoms

Draw polyhedra passing through bonded atoms to selected atoms and display triangles for selected atoms bonded to exactly three other atoms.

Controls

Show polyhedra & triangles toggle Made polyhedra and triangles visible.
Color by center atom toggle Instead to have polyhedra colored by “Surface color” choice, color them by central atom color.
Select central atoms by radio buttons Select the central atoms by one of the following ways:
Symbol
The atom type symbol.
Label
The atom label eventually present in the input structure.
Index
Index of the atom inside the structure. The index starts from zero.
All
Select all atoms.
Central atoms selector text Space separated list of atoms selectors. The value is taken after exiting the field or pushing the Enter key.
Opacity slider Opacity of the polyhedra facies and triangles if “Color by center atom” is set.
Surface color color selector Select the color of the polyhedra and triangles if “Color by center atom” is not set.
Constrain vertex count radio buttons Constrains the number of atoms composing a polyhedra:
Any
Three or more connected atoms.
Exact
The connected atom count should be exactly the value set below.
At least
The connected atom count should be at least the value set below.
Vertices slider Number of polyhedra vertices for the constrains above. It is visible if “Constrain vertex count” is not set to “Any”.
