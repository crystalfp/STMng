Display Unit Cell

Description

Displays the borders of the unit cell. Nothing is displayed if the molecular data does not contain unit cell data.

Input Ports

structure_in MoleculeType Molecular structure data containing unit cell definition.

Parameters

Black toggle If set draw the unit cell borders black, otherwise white.
Shift origin back toggle If set moves the origin of the unit cell from the center of the true unit cell back to the correct origin.
Visible toggle Toggle unit cell visibility.

Output Ports

fld Mesh+Node_Data The unit cell as a Line cell. The node data is set to RGB white or black.
obj DefaultObject The renderable version of fld.
