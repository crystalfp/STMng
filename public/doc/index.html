STMng documentation

See The Molecule new generation (STMng) is a visualization tool that implements some of the STM4 functionalities.
Modules

Structure Reader: read the file containing one or more atomic structures

Draw Structure: render an atomic structure

Draw Unit Cell: render an atomic structure

Draw Polyhedra: draw a convex hull around atoms

Chart Dialog: open a window showing a chart

Viewer 3D: the viewer frame

Capture view: capture the viewer content as snapshot or movie

Compute symmetries: find and apply symmetries to the structure

Compute bonds: add bonds to the given structure

Project

This is the description of the modules used and their interconnections.

The project or default project will be automatically loaded on application start. The project file could be loaded on the command line.

A project is a JSON formatted file with extension .stm with the following content:

  • graph List the loaded nodes and their interconnections
  • viewer Optional section to collect the viewer parameters
  • ui Optional section to collect the parameters for each node
  • currentId Optional parameter to set the initial ui panel to this node key.

Graph content

This section is a JSON record with key an unique identifier of the module

The graph record content

  • label: The label that appear in the UI selection panel
  • type: The type of node, that is its functionality.
  • in: (optional) The id of the node from which this node takes the input. It is missing if the node has no input or it the input come from the graphical scene.
Shortcuts

System menu

  • Alt+F: File submenu
  • Alt+V: View submenu
  • Alt+H: Help submenu

Project file group

  • Control+O: Load a project file
  • Control+D: Load the default project
  • Control+S: Save the project to the file from which has been loaded
  • Control+Shift+S: Save the project to a selected file. This file will be loaded automatically next time

Application group

  • Control+R: Reload the application
  • Control+Shift+R: Force reload the application
  • Control+Q: Quit the application without asking confirmation
  • F11: Toggle full screen
  • Control+T: Toggle the dark theme for the application
  • Control+E: Toggle the viewer extension, that is hide the control panel on the left
During development the following keys are also available:
  • F12: Toggle development tools
  • Control+F12: Toggle development tools also on opened secondary windows

Help

  • F1: Open the STMng help in browser
  • Control+A: Open the About dialog
Interaction with the viewer

Interacting with the viewer means moving the camera around the visualized structure.

This interaction is done through mouse movements and keyboard arrows.

Terminology

  • Rotate: Rotate the camera around the scene center
  • Pan: Move the camera on the viewer plane
  • Zoom: Change the magnification of the camera without moving it
  • Dolly: Move the camera perpendicular to the viewer plane

Mouse

  • Drag with left mouse button: Rotate
  • Drag with right mouse button: Pan
  • Drag with center mouse button: Dolly
  • Mouse wheel: Zoom

Keyboard arrows

  • Any arrow: Pan
  • Control + any arrow: Rotate
  • Shift + any arrow: Zoom (Left and Down do the same, as do Right and Up)
  • Alt + any arrow: Dolly (Left and Down do the same, as do Right and Up)
