STMng — Program editor documentation

The loaded project structure could be seen and changed using the File → Project editor menu entry. This entry opens the Project editor in its own window depicted below.

The window title contains the filename of the currently loaded project or the “default project” mark.

The leftmost column, labelled “Available nodes”, list all the nodes types provided by the application.

Nodes from this column could be dragged to the graph chart on the right to create a new node in the project. In the name a string in parenthesis mark these created nodes.

Clicking a node in the graph chart marks it as selected with a yellow background. Clicking on the chart background deselects it. The selected node show its data in the top right panel. The label of the node could be edited in the panel. With a button the node could be also deleted after a confirmation dialog.

The nodes’ positions could be changed by dragging the nodes around.

The connection between nodes is done clicking on the output port (the rightmost one) and when the cursor becomes a black cross , dragging it to a free input port (the left one). If a connection is redone, it is deleted.

Going to one end of the connection the cursor changes to then this end of the connection can be drag to another connection point.

At the end, “Save modified project” asks for a project filename (.stm), save here the modified project and load it in the application. Closing the window with the “Close” button loses the changes.
