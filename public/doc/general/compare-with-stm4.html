STMng — functionalities comparison with STM4

At the root, the difference between STMng and its predecessor STM4 is that the latter is built on top of the commercial tool AVS/Express and the former uses open tools and languages.

AVS/Express is a comprehensive and versatile data visualization tool designed for sophisticated problem-solving in fields like science, engineering, medicine, telecommunications, and environmental research. In this environment STM4 is one of the possible applications. Instead STMng is an application focused on one goal only: visualize and analyze chemistry and crystallographic data.

Going deeper, the main difference lies in the underlying architecture. STM4 uses the Object Manager from AVS/Express that manages a DAG of modules. STMng instead is based on a simpler linear assembly of modules. If you open the project editor you immediately see this.

Also AVS/Express supports the creation of macro modules that combine modules into super-modules. STMng instead has only the project as the unit of customization.

The following tables comes directly from the STM4 documentation and have been augmented with references to the corresponding functions in STMng. At the end there is a list of STMng functionalities not present in STM4.

The STM4 modules grouped by function:

Readers Writers Readers and writer for various, static and dynamic, chemical file formats plus movies and images production.
Bonds Bonds computation, editing and management.
Modules Modules that compute derived structures.
Selection Selector modules to extract a subset of the atoms.
Trajectories Modules related to dynamic data.
Crystallography Unit cells, periodic lattices and symmetries support.
Volume Rendering of volumetric data like Gaussian cubes.
Display Rendering of chemical structures and other graphical objects.
Probe and Measure Interact with the structures: measure and editing.

Readers Writers

Read Structure Read structural and volumetric chemical data. Structure Reader node.
Read Structure No Bonds Read a structure from file without computing bonds. Structure Reader with removing or disabling the subsequent Compute bonds node.
Read Scalar Read a set of scalar values from a file. The reader for a column of scalars representing energies is already included in the Structure Reader node.
Read Vector Read a set of 3D vector values from a file. In the current architecture there is no place for vector values.
Make Movie Capture a sequence of frames, one for each different viewer content, and assemble them in a movie (AVI or MPEG). Capture view node. Note that the movie formats offered are the newer ones (MP4, Matrioska, WebM) and not the legacy ones.
Output Image Save a snapshot of the content of the current viewer. Capture view node. Also PDF is offered as an output format.
Write Structure Write a single structure or a trajectory to a chemistry data file in the selected format. Structure writer node.
Write POV Ray Write the current view as a POV-Ray raytracer scene. Maybe could be of interest to export scene for the Persistence of Vision Raytracer. Or could be more interesting the WebXR augmented reality support.
Read Fermi Surface Read a file in EIGENVAL format and output a volume that, using isosurfaces, display the Fermi surfaces at a given energy level.
Read Fermi Bands Read a file in EIGENVAL format and output for all bands their energy along a given polyline.
Write Summary Write a summary of the series of structures given as input. A summary of the current structure is available in the Measures node although it is not exported and it does not cover a sequence of structures loaded.

Bonds

Compute Bonds Compute atomic bonds. Compute bonds node.
Count Bonds Add as atom data the atom's number of bonds. Draw structure could color atoms by their number of bonds.
Count H Bonds Count the number of H Bonds present in the input structure. The count is added to the structure global data. A count of the current structure bonds and H bonds is available in the Measures node.
Compute Selected Bonds Adds bonds to specific atoms using a distance criteria. It is not entirely clear what is the original goal of this module.
Set Manual Bonds Manually add bonds between atoms. It is not entirely clear what is the original goal of this module.

Modules

Solvent Excluded Surface Compute the molecule Solvent Excluded Surface.
Backbone Compute a protein backbone. Structure backbone node.
General Backbone Display a tube through user selected atoms. Structure backbone node.
Vector Glyph Display with arrows the vector data associated with a given structure. Currently there is no provision for vector values associated to the atoms.
Merge Values Add a new scalar or vector data to atoms, bonds or globally to the given structure. Currently there is no provision for values associated to the atoms.
Atoms Values Trasp Atoms are rendered with transparency dependent on the associated scalar value Currently there is no provision for values associated to the atoms.
Particle Density Compute the average particle mass density using uniform binning.
Interpolating Plane Build the least square plane interpolating a set of picked atoms. What could be done with the interpolated plane?

Selection

Select Atoms Select specific atoms. What could be done with the selected atoms?
Select Fragments Select specific structure fragments (i.e. residues). What could be done with the selected atoms?
Threshold Data Select atoms whose associated scalar value meets various criteria. No scalar value associated to atoms.
Delete Atoms Interactively delete atoms.

Trajectories

Accumulate Traces Accumulate traces (accumulated positions) for specific atoms in a sequence of time steps. Draw trajectory node.
Accumulate Global Data Accumulate a global scalar data in a form suitable for charting. No scalar value associated to atoms.
Interpolate Sequence Linearly interpolate a given number of positions between each pair of timesteps. With the current architecture the implementation of a similar module is not simple.
Velocity Mean Compute the average particle velocity using uniform binning.
Enthalpy Transitions Search a range of pressure values to find where the structure with the minimum enthalphy changes.

Crystallography

Replicate Lattice Replicate the structure unit cell along the three base vectors. Draw unit cell node.
Apply Symmetries Compute new atoms positions from the space group of the input molecule. Compute symmetries node.
Shift Unit Cell Shift the unit cell to re-center periodic phenomena that are split between the opposite sides of the cell. Draw unit cell node.
Enlarge Unit Cell Add atoms outside the unit cell by different criteria. Compute bonds node.
Move Structure Translate and rotate a given structure modifying all the atoms coordinates. Not very useful for a single structure visualized. Something similar is available from the Compute fingerprints Compare selected window.
Crop Structure Crop a structure and the optional volume data inside a given geometry. Slice structure node.
Crop From Plane Remove atoms that dist more than a given value from a plane. Slice structure node.
Miller Plane Crop the atoms using the plane identified by given Miller indices. Slice structure node.
Draw Polyhedra Display a convex polyhedron around selected atoms. Draw polyhedra node.
Draw Simple Polyhedra Display a convex polyhedron around selected atoms using a different setup. Draw polyhedra node.
X Ray Structure Factor Compute and display simplified X-Ray structure factors. Compute X-Ray diffraction node.
Remove Border Atoms Remove atoms that bonds with atoms outside the unit cell. Not clear the goal of this module.
Merge Unit Cell Apply the unit cell of one structure to another one. Not clear the goal of this module.
Find Symmetries Find symmetries in the input structure and simplify it. Compute symmetries node.

Volume

Isosurface Create an isosurface for a given scalar volumetric value. Isosurface node.
Orthoslice Create an orthoslice for a given scalar volumetric value. Orthoslice node.
Volume Render Volumetric render of a scalar valued volume. Need to understand how to implement volume rendering.
Cube on Surface Assign to a surface the values of the volume in which the surface is immersed in the positions occupied by the surface itself. It is like a free orientation orthoslice. Instead a generic surface is not possible.
Cut Surface Cut a surface (eg. a Solvent Excluded Surface) with a plane. It is not clear the goal of this module.
Bicolor Surface Color a surface on one side with a user specified color leaving the other one with its original coloring. It is not clear the goal of this module.
Interpolate Volume Interpolate a given scalar volume adding points between original grid points. Interpolate volume node.

Display

Draw Structure Transform the molecule structure data into graphic objects for display. Draw structure node.
Draw Sw Structure Transform the molecule structure data into graphic objects for display using a software render method. Draw structure node but note that there is no software render in STMng.
Info Sets Editor Manage, edit, create and destroy Info Sets. This will appear when editing of the atoms characteristics will be implemented.
Display Unit Cell Displays the borders of the unit cell. Draw unit cell node.
Set Rendering Mode Set the rendering mode on an atom by atom base. Not useful.
Axis Glyph Display three arrows to mark the origin and the XYZ axis directions, or the unit cell abc vectors. The glyph could also be offset from the origin. Viewer 3D helpers, but they cannot be offset from the origin. Note that there are also plane orientation helpers. Also the Display unit cell node could show the cell basis vectors.
Background Fade Provide a colored backdrop for which you can set the four corners colors. Not useful.
Logo Create a logo image to be added to a viewer. The image is locked in the viewer and is not transformed by standard mouse actions. Is it useful?
Enclosing Ellipsoid Compute an ellipsoid that encloses the given set of atoms.
Time Legend Display a progress bar to show the current timestep / current time for an animated molecule trajectory. The progress bar is in the Structure reader node. But for a movie could be useful to have it in the viewer.
Color Legend Display a legend for the structure representation colors. It is visualized from specific nodes.
Discrete Legend Color legend for integer values in an user specified range. It is visualized from specific nodes.
Smooth Tube Create a tube that connect all the given points. The Structure backbone node draws a tube through selected atoms.
Error Tube Create a tube with a radius proportional to the values associated to the connected points. Currently there is no provision for scalar values associated to the atoms.
Structure Info Show count of the number of atoms present in the input structure. Measures node.

Probe and Measure

Measure Structure Access information about and measure a displayed structure. Measures node.
Measure Closer Atoms Measure intra-atomic distances between a given atom and the surrounding ones in a given range. Measures node gives distances for bonded atoms.
Pick Unit Cell Redefines the unit cell for the displayed structure by picking on four atoms. It is not clear the goal of this module.

Exclusive functionalities in STMng

Compute fingerprints Compute fingerprints for a set of structures.
Find similar structures Match a structure against the “prototype” and “collection” structures.
Structure reader special collections Collection of model structures and prototypes.
Variable composition Split compositions and remove duplicates structures.
Capture view Capture the viewer content also as an STL model for 3D printing.
Compare structures Visually compare pair of structures.
Fingerprints energy landscape Show the energy surface interpolating the computed fingerprints.
Fingerprints scatterplot Show the fingerprints as points in a 2D space.
Show trajectory displacements Show atom’s trajectory average positions and displacements.
Variable composition convex hull Show enthalpy of formation for the structures versus they compositions.
