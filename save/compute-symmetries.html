Compute Symmetries

Description

Find and apply symmetries to the input structure. This node has two parts: Find symmetries and Apply symmetries.

Find symmetries

This part tries to determine the input structure symmetries using the KPLOT program. Note that this program has limits on the size of the input structures.

Apply symmetries

This part apply the structure symmetries removing duplicated atoms and filling the unit cell if needed.

Parameters

Enable find symmetries switch Enable the execution of the KPLOT program, otherwise the input structure is simply copied to the apply symmetries part.
Ignore input symmetries switch Ignore the input symmetries. Needed when the symmetries are not given with a space group.
Tolerances — S slider Tolerance for symmetry centers. Input to the KPLOT program.
Tolerances — T slider Tolerance for translations. Input to the KPLOT program.
Tolerances — G slider Tolerance for rotations. Input to the KPLOT program.
Enable apply symmetries switch Apply the symmetries to the input structure, otherwise the input structure is simply copied to the output. Above this option the symmetries that will be applied are shown.
Fill unit cell switch Fill the unit cell with the atoms that appear on the unit cell boundaries.
Show symmetries dialog button Opens a dialog that show symmetries from the input structure and output structure symmetries.
